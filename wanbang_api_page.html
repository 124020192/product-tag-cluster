      万邦taobao API 接入说明

      1. API地址:
        https://api-gw.onebound.cn/taobao/

      2. 调用示例：
        https://api-gw.onebound.cn/taobao/item_search/?key=& &q=女装&start_price=0&end_price=0&page=1&cat=0&discount_only=&sort=&page_size=&seller_info=&nick=&ppath=&imgid=&filter=&&lang=zh-CN&secret=

      3. 参数说明

        • 通用参数说明
          • 参数不要乱传，否则不管成功失败都会扣费
          • url说明 https://api-gw.onebound.cn/平台/API类型/ 平台：淘宝，京东等， API类型:[item_search,item_get,item_search_shop等]
          • version:API版本
          • key:调用key,测试key:test_api_key
          • secret:调用secret,测试secret:(不用填写)
          • cache:[yes,no]默认yes，将调用缓存的数据，速度比较快
          • result_type:[json,xml,serialize,var_export]返回数据格式，默认为json
          • lang:[cn,en,ru] 翻译语言，默认cn简体中文
          • secret:密钥

        • API:item_search 参数说明：

          • q:搜索关键字
          • cat:分类ID
          • start_price:开始价格
          • end_price:结束价格
          • sort:排序[bid,bid,bid2,_bid2,_sale,_credit]
            (bid:总价,bid2:商品价格,sale:销量,credit信用,加            前缀为从大到小排序)
          • page:页数
          • page_size:每页宝贝数量，默认40
          • seller_info:是否获取商家信息[yes,no],默认yes

        • API:item_get 参数说明: num_iid:宝贝ID
